# OOP
Здесь собраны четыре Лабораторной работы по Объектно-ориентированному программированию в МГТУ "Станкин". В каждой папке, соответствующей лаборотоной работе, представлен файл с расширением .cpp, для наглядности кода. В Лабороторных работах может быть несколько заданий, поэтому в первой и третьей указаны несколько файлов, назания соответсвтуют номеру задания.
